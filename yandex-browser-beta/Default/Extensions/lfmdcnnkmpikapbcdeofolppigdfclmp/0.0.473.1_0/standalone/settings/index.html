Настройки: Яндекс.Музыка
Чтобы изменить настройки горячих клавиш:

1. В адресную строку браузера введите . Откроется страница настройки «Горячих клавиш».

2. Найдите блок «Яндекс.Музыка» и задайте нужные клавиши или их сочетания для команд.
